> **Внимание**: Этот проект создан с помощью AI (Claude Code / Opus 4.6). Все инструменты соответствуют официальному Proxmox API, код проверен, но **всегда проверяйте команды перед выполнением на боевой инфраструктуре**. Система безопасности помогает, но не заменяет понимание того, что вы делаете. Думайте прежде чем нажать Enter.

---

Не просто обёртка над API — **поведенческий тулкит**, который учит AI-ассистентов правильно управлять Proxmox инфраструктурой:

- Создавать VM с правильным BIOS (SeaBIOS для Linux, OVMF для Windows)
- Настраивать Docker-ready LXC контейнеры с nesting и apparmor
- Проверять безопасность перед каждым деплоем
- Диагностировать проблемы системно (а не "просто перезагрузи")
- Защищать от случайного удаления с обязательным подтверждением

Работает с **Claude Code** | **Gemini CLI** | **OpenAI Codex** | **Cursor**

## Быстрый старт

### 1. Добавьте в проект

Создайте `.mcp.json`:

```json
{
  "mcpServers": {
    "superproxmox": {
      "command": "npx",
      "args": ["super-proxmox"],
      "env": {
        "PVE_URL": "https://ВАШ-IP:8006",
        "PVE_USER": "root@pam",
        "PVE_PASSWORD": "ваш-пароль"
      }
    }
  }
}
```

### 2. Перезапустите AI-ассистент

Готово. 141 инструмент доступен.

### Полный стек (PVE + PBS + PDM)

```json
{
  "mcpServers": {
    "superproxmox": {
      "command": "npx",
      "args": ["super-proxmox"],
      "env": {
        "PVE_URL": "https://10.0.0.1:8006",
        "PVE_USER": "root@pam",
        "PVE_TOKEN_ID": "mcp",
        "PVE_TOKEN_SECRET": "xxx",

        "PBS_URL": "https://10.0.0.2:8007",
        "PBS_USER": "root@pam",
        "PBS_TOKEN_ID": "mcp",
        "PBS_TOKEN_SECRET": "yyy",

        "PDM_URL": "https://10.0.0.3:443",
        "PDM_USER": "admin@pdm",
        "PDM_TOKEN_ID": "mcp",
        "PDM_TOKEN_SECRET": "zzz"
      }
    }
  }
}
```

## Переменные окружения

| Переменная | Обязательна | Описание |
|-----------|-------------|----------|
| `PVE_URL` | Да | URL Proxmox VE (`https://IP:8006`) |
| `PVE_USER` | Нет | Пользователь (по умолчанию `root@pam`) |
| `PVE_PASSWORD` | * | Пароль |
| `PVE_TOKEN_ID` | * | ID API-токена |
| `PVE_TOKEN_SECRET` | * | Секрет API-токена |
| `PBS_URL` | Нет | URL Proxmox Backup Server |
| `PDM_URL` | Нет | URL Proxmox Datacenter Manager |

\* Нужен либо пароль, либо пара token ID + secret.

## Система безопасности

**17 деструктивных операций** защищены обязательным подтверждением.

Перед выполнением AI обязан:
1. **Сказать** что хочет сделать и зачем
2. **Показать** что именно будет уничтожено
3. **Проверить** существует ли бэкап
4. **Спросить** явное подтверждение
5. **Ждать** ответа "да" от пользователя
